# Api Rest

Este proyecto es una Api Rest de Cafeteria Villa Café, la parte Backend ha sido creada en Node JS y con base de datos en MongoDB.

## Scripts disponibles

En el directorio del proyecto, puedes ejecutar

* Para iniciar MongoDB abre una terminal y ejecuta el comando: mongod 
* Para ejecutar el proyecto abre una terminal y ejecuta el comando: npm start


## Ejecutar la aplicación en modo desarrollo.
  Abre http://localhost:5000 para verla en el navegador.

# Dependencias instaladas:
- Node JS
- Express
- MongoDB
- Crypto-js
- Dotenv
- Jsonwebtoken
- Mongoose
- Nodemon
- Stripe
